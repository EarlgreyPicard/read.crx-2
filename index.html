read.crx 2

read.crx 2はGoogle Chromeアプリとして作られた2chブラウザです。 2ch及びその互換BBS、まちBBS、したらばの閲覧に対応しています。

一般の利用者向けの配布はこちら

ビルド手順

npm, Bundler, ImageMagickが予め導入されている必要が有ります。

git clone --recursive git://github.com/awef/read.crx-2.git

cd read.crx-2

npm install
bundle install

bundle exec rake

商用利用時の注意

read.crx 2のソースコードはMITライセンスですが、read.crx 2がアクセスするサービスの中には商用利用に制限が存在する場合が有ります。ご注意下さい。
